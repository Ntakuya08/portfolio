SmartPhone UI

  • 構成 : トップページのみ
  • 制作期間 : 30時間程度
  • 使用ツール : HTML / CSS / jQuery / Photoshop / XD

架空のレストランのウェブサイトです。トップぺージのみですが、Adobe XDを使用してデザインカンプから作成しました。制作期間はデザインカンプが2日、サイト自体が3日ほどです。カンプをできる限り詳細に制作したのでHTMLコーディングはスムーズに行えました。デザインはシンプルに見やすい形にまとめられたかと思っています。また、どのような画面サイズでも問題なく表示できるようレスポンシブ対応には力を入れました。

カンプを実寸より大きいサイズで作ってしまったため、フォントサイズや余白をカンプの通り設定できず調整に時間を食われたのが反省点です。jQueryを利用しましたがまだ理解が浅く、思うように動作せずその点でも時間を使ってしまいました。
